# Animal Inside(A-IN)

반려동물을 키우는 사람들을 위한 커뮤니티 프로젝트

## 📁 프로젝트 소개

반려동물 보호자들을 위한 소셜 네트워크 서비스로 사용자들은 24시간 동안 유지되는 스토리 기능을 통해 반려동물의 일상을 공유하고, 실시간 채팅과 게시글을 통해 다른 보호자들과 소통하는 공간입니다.
